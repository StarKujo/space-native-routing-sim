# scripts/ 说明

## 用途

存放论文生成、修订、验证的工具脚本，是项目的文稿工作流引擎。

## 文件列表

### Python 脚本

| 文件 | 功能 | 输入 | 输出 |
|------|------|------|------|
| `create_papers.py` | 批量生成1篇总结+8篇论文初稿 | 代码内嵌内容 | `../new/` 目录 |
| `refine_revised_papers.py` | 对修订稿进行XML级段落精修 | `../revised_drafts/*.docx` | 原地修改 |
### PowerShell 脚本

| 文件 | 功能 |
|------|------|
| `rewrite_selected_papers.ps1` | 重写03/05/08三篇论文（全新内容） |
| `enhance_selected_papers.ps1` | 对选中论文进行内容增强 |
| `fix_05_intro.ps1` | 修复论文05的引言部分 |
| `repair_incomplete_paragraphs.ps1` | 修复DOCX中的不完整段落 |
| `_verify_docs.ps1` | 检查文档初稿禁用语残留 |

### 启动器

| 文件 | 功能 |
|------|------|
| `launch claude.bat` | 切换到项目目录并启动Claude CLI |
| `launch codex.bat` | 切换到项目目录并启动Codex CLI |

## 环境依赖

- Python 3.x + `python-docx` 库
- PowerShell 5.1+（Windows原生）
- Microsoft Word（用于查看DOCX）
